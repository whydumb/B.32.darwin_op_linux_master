Do not run this specific controller under SSH client windows (i.e. PuTTY). 
The controller will not run and will display a segmentation error.

To run this controller:

-Open connect to DARwIn-OP via remote desktop (i.e. VNC)
-Open a terminal window
-Compile and run the controller
-A keyboard-related "Webots cross-compilation keyboard input "sub-window will pop up
-During the simulation make sure you've selected said sub-window
